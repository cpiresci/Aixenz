Política de Privacidade

Em conformidade com a LGPD (Lei 13.709/2018) e regulamentações do BACEN
01Quem Somos e Escopo

O AIXENZ é uma plataforma de inteligência financeira por Inteligência Artificial operada por AIXENZ Tecnologia Financeira Ltda., sediada no Brasil. Esta Política de Privacidade descreve como coletamos, usamos, armazenamos e protegemos os dados pessoais de nossos usuários, em conformidade com a Lei Geral de Proteção de Dados (LGPD — Lei 13.709/2018) e demais regulamentações aplicáveis.

Esta política se aplica a todos os usuários do aplicativo AIXENZ (Android e iOS), do site aixenz.com e de qualquer serviço associado à plataforma AIXENZ.

02Dados que Coletamos

Coletamos apenas os dados estritamente necessários para a prestação dos nossos serviços:

  • Dados de identificação: nome completo, CPF, data de nascimento, endereço
  • Dados de contato: e-mail, número de telefone
  • Dados financeiros: informações de renda, patrimônio, perfil de investidor declarado pelo usuário
  • Dados de uso: histórico de análises realizadas, interações com os 16 agentes de IA
  • Dados de dispositivo: sistema operacional, versão do app, identificador do dispositivo (para segurança)
  • Dados de transação: quando aplicável, informações de pagamento processadas via Stripe (não armazenamos dados de cartão — Stripe é o responsável pelo processamento PCI-DSS)
  • Dados de localização aproximada: país/estado, para conformidade regulatória e personalização
03Como e Por Que Usamos Seus Dados

Seus dados são utilizados exclusivamente para:

  • Prestação do serviço: gerar análises financeiras personalizadas pelos 16 agentes de IA
  • Segurança: detectar fraudes, acessos não autorizados e proteger sua conta
  • Compliance regulatório: cumprir obrigações perante o BACEN, LGPD e regulamentações internacionais aplicáveis
  • Melhoria do produto: análise agregada e anonimizada de uso para aprimoramento dos algoritmos
  • Comunicação: envio de notificações relevantes sobre a sua conta (não fazemos spam)
  • Faturamento: processamento de pagamentos via Stripe

Não vendemos, alugamos ou compartilhamos seus dados pessoais com terceiros para fins de marketing. Seus dados financeiros nunca são usados para treinar modelos de IA de terceiros sem seu consentimento explícito.

04Inteligência Artificial e Decisões Automatizadas

O AIXENZ utiliza 16 agentes de Inteligência Artificial para análise financeira. É importante entender:

  • As análises geradas pelos agentes de IA têm caráter exclusivamente informativo e não constituem recomendação de investimento regulamentada
  • Todas as decisões dos agentes são registradas em nosso XAI Audit Log (Explainable AI) e podem ser consultadas pelo usuário
  • Os dados enviados ao prompt dos agentes são processados pelos provedores de LLM (SambaNova, OpenRouter, Google Gemini, Cerebras) conforme suas próprias políticas de privacidade
  • Não utilizamos seus dados para treinar modelos de terceiros sem consentimento
  • O usuário tem o direito de questionar qualquer decisão automatizada que o afete
05Compartilhamento de Dados com Terceiros

Compartilhamos dados apenas quando estritamente necessário e com garantias contratuais de proteção:

  • Stripe: processamento de pagamentos (PCI-DSS Level 1). Não armazenamos dados de cartão
  • Provedores de LLM: SambaNova AI, OpenRouter, Google (Gemini), Cerebras — apenas os dados do prompt de análise, sem identificação pessoal direta
  • Hostinger / Render: infraestrutura de hospedagem com acordo de processamento de dados (DPA)
  • BACEN e autoridades regulatórias: quando exigido por lei ou regulamentação
  • Parceiros BaaS (Bankly, Wise, Dock — quando disponíveis): dados mínimos necessários para execução de transações financeiras, com consentimento explícito
06Armazenamento e Segurança

Adotamos as seguintes medidas técnicas e organizacionais de segurança:

  • Criptografia em trânsito: TLS 1.3 em todas as comunicações
  • Autenticação: JWT com expiração configurável, bcrypt para senhas
  • Banco de dados: MySQL na infraestrutura Hostinger com backups diários automáticos
  • Zero Trust: cada chamada de API é autenticada e autorizada individualmente
  • Retenção de dados: dados de análise são mantidos por até 5 anos para fins de auditoria regulatória; dados de conta são excluídos dentro de 30 dias da solicitação
  • Logs de auditoria: mantidos por 7 anos conforme exigência do BACEN
07Seus Direitos (LGPD)

Como titular de dados, você tem os seguintes direitos garantidos pela LGPD:

  • Acesso: solicitar cópia de todos os seus dados pessoais que processamos
  • Correção: corrigir dados incompletos, inexatos ou desatualizados
  • Anonimização ou exclusão: solicitar a anonimização ou exclusão dos seus dados (sujeito a obrigações legais de retenção)
  • Portabilidade: receber seus dados em formato estruturado e interoperável
  • Revogação de consentimento: retirar consentimento a qualquer momento
  • Oposição: opor-se ao processamento em casos específicos
  • Informação sobre compartilhamento: saber com quem compartilhamos seus dados

Para exercer qualquer desses direitos, entre em contato pelo e-mail: privacidade@aixenz.com. Respondemos em até 15 dias úteis.

08Cookies e Tecnologias de Rastreamento

O AIXENZ utiliza tecnologias de armazenamento local de forma mínima e transparente:

  • LocalStorage: armazena token de autenticação JWT e preferências de idioma — necessário para o funcionamento do serviço
  • Sem cookies de rastreamento: não utilizamos cookies de terceiros para publicidade ou rastreamento comportamental
  • Sem pixels de rastreamento: não incorporamos pixels de Facebook, Google Ads ou similares
09Transferências Internacionais de Dados

Ao utilizar provedores de LLM internacionais (SambaNova — EUA, Google Gemini — EUA, Cerebras — EUA, OpenRouter — EUA), os dados do prompt de análise podem ser processados fora do Brasil. Garantimos que:

  • Apenas dados necessários para a análise são enviados, sem identificação pessoal direta quando possível
  • Todos os provedores possuem políticas de privacidade compatíveis com padrões internacionais (GDPR, CCPA)
  • Contratos de processamento de dados (DPA) estão estabelecidos quando aplicável
10Menores de Idade

Os serviços do AIXENZ são destinados exclusivamente a maiores de 18 anos. Não coletamos intencionalmente dados de menores. Se identificarmos que coletamos dados de menores sem consentimento parental, excluiremos imediatamente tais informações. Caso tenha conhecimento de tal situação, contate-nos em privacidade@aixenz.com.

11Alterações nesta Política

Esta política pode ser atualizada periodicamente. Notificaremos os usuários sobre mudanças significativas via e-mail cadastrado e/ou notificação no aplicativo com no mínimo 15 dias de antecedência. O uso continuado dos serviços após a notificação implica aceite das alterações.

O histórico de versões desta política estará disponível em aixenz.com/privacidade.
